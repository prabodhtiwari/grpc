# grpc

> Grpc dev

grpc service example using Protobuf 3

Built on
--------

* **[golang](https://golang.org)**
* **[gRPC](http://www.grpc.io)**
* **[protobuf](https://developers.google.com/protocol-buffers/)**

Install
-------

```bash
    git clone git@github.com:prabodhtiwari/grpc.git
    go get
```

Run Server
----------

```bash
    ./server.sh
```

Run Client
----------

```bash
    ./client.sh
```
